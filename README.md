# UD1-Project-By-TheAnimalScientists
Created by: Jeffrey Silvas, Katelyn Ho, Noah Vargas Polo

The Giraffe Research Program is an application designed to educate users on the giraffe species and populations. It contains an assortment of facts and statistics, as well as functionality to make relevant calculations and averages.
How It Works: 
The user is prompted with a list of choices about facts and information from these giraffes, and the user chooses one of them.

Katelyn created the static methods for printing the scientific names of giraffes and fun facts about them.
Jeffrey created the static methods for printing the endangered status of various giraffe species and anatomical facts.
Noah created the static methods for calculating the projected population growth and the population decline of giraffes.
